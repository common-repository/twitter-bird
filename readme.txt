Twitter Bird will display how many people follows you on your Twitter account.


== Description ==
Twitter Bird is a free plugin, aim to display total number of twitter followers of your twitter account.
When mouse over on the bird image, this Twitter Bird will display how many people follows you on your Twitter account.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload Plugin in '/wp-content/plugins/' directory,
2. Activate plugin through the 'Plugins' menu in WordPress
3. Goto Twitter Bird Settings page and enter your Twitter ID
4. Place `<?php echo gotwitterbird(get_option('twitteraccount')); ?>` in your template,
